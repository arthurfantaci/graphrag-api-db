# Plan Command

Create an implementation plan for: $ARGUMENTS

## Instructions

1. Read relevant source code before proposing changes
2. Check CLAUDE.md for project conventions
3. Check docs/ for existing analysis or prior plans

## Plan Structure

Write a Markdown plan that includes:
- **Problem**: What needs to change and why
- **Implementation steps**: Ordered list with file paths and line numbers
- **Tests**: What tests to write or update
- **Verification**: How to confirm the change works

## Output

Save the plan to `docs/` and provide a summary.
